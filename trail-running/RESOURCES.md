# Trail Running Resources

## Knowledge

- [Official route: XTERRA Dolomiti di Brenta Trail SHORT 21K](https://www.dolomitidibrentatrail.it/en/routes/xterra21k/)
  Canonical race constraint. Use for: distance, D+, start time, 5h maximum time, 3h Croz dell'Altissimo cutoff, and route sequence.
- [`../trail-brenta-2026/data/race.json`](../trail-brenta-2026/data/race.json)
  Local structured version of the race goal and 2025 route profile. Use for: Brenta-specific lesson framing.
- [`../trail-brenta-2026/data/plan.json`](../trail-brenta-2026/data/plan.json)
  Local training plan phases and constraints. Use for: matching lessons to the current phase and avoiding advice that conflicts with the plan.
- [iRunFar: An Introduction to Powerhiking](https://www.irunfar.com/an-introduction-to-powerhiking)
  Practical explanation of power hiking as purposeful pacing, not a walk break. Use for: uphill technique and run/hike transitions.
- [iRunFar: Going Up - An Introduction to Uphill Trail Running Techniques](https://www.irunfar.com/going-up-an-introduction-to-uphill-trail-running-techniques)
  Uphill trail technique, including line choice and efficiency on loose/rocky terrain. Use for: climbing mechanics and foot placement.
- [iRunFar: An Introduction to Trail Racing, Part 1](https://www.irunfar.com/an-introduction-to-trail-racing-part-1-pre-race-preparation)
  Race preparation basics, including course knowledge, mandatory kit, and strategy. Use for: pre-race execution planning.
- [TrainingPeaks: Beginner's Guide to Trail Running](https://www.trainingpeaks.com/blog/beginners-guide-trail-running/)
  Broad beginner overview. Use for: differences between road and trail running, etiquette, gear, and pacing mindset.
- [Appalachian Mountain Club: How to Run Downhill](https://www.outdoors.org/resources/amc-outdoors/outdoor-resources/how-to-run-downhill-mastering-trail-runnings-most-critical-skill/)
  Conservative downhill technique advice from outdoor educators. Use for: safe descending, gaze, arm balance, and impact control.
